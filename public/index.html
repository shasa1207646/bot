⚡ Slash-команды

/help

Показывает список всех команд бота.

/help

/ban <user> [reason]

Банит пользователя. Требует право BanMembers.

/ban @User спам

/kick <user> [reason]

Кикает пользователя. Требует право KickMembers.

/kick @User нарушение правил

/create-role <name> [color]

Создаёт роль с указанным hex-цветом. Требует ManageRoles.

/create-role Модератор #ff0000

/action <user> give_role <role>

Выдаёт пользователю роль по названию. Требует ManageRoles.

/action @User give_role VIP

/action <user> send_message <text>

Отправляет пользователю личное сообщение.

/action @User send_message Добро пожаловать!

ℹ️ О боте

Discord-бот на Node.js + TypeScript + discord.js v14 с веб-панелью на Express 5. Система заявок, фильтр слов, slash-команды.

🔧 Подключение Google Forms

Создайте форму в Google Forms с нужными полями (Discord ID, возраст и т.д.)
В форме откройте Инструменты → Редактор скриптов
Вставьте скрипт ниже, замените ВАШ_ДОМЕН на URL вашего бота
В редакторе: Триггеры → Добавить триггер → onFormSubmit → При отправке формы
Готово! Теперь каждая отправка формы будет отправлять заявку боту.
var BOT_WEBHOOK_URL = "https://ВАШ_ДОМЕН/api/applications"; var FIELD_MAP = { discord_id: "Дискорд", discord_username: "Ник", age: "Сколько вам лет", name: "Ваше имя (если не хотите писать напишите свой ник)", activity: "Сколько в день вы актив", games: "В какие игры вы играете (не обязал)", rules: "Принимаете ли вы правила сервера?" }; function onFormSubmit(e) { var items = e.response.getItemResponses(); var payload = {}; items.forEach(function(item) { var q = item.getItem().getTitle(); for (var key in FIELD_MAP) { if (q.indexOf(FIELD_MAP[key]) !== -1) { payload[key] = item.getResponse(); } } }); var options = { method: "post", contentType: "application/json", payload: JSON.stringify(payload), muteHttpExceptions: true }; UrlFetchApp.fetch(BOT_WEBHOOK_URL, options); } function testConnection() { var testData = { discord_id: "123456789", discord_username: "TestUser", age: "20", name: "Тест", activity: "4 часа", games: "CS2", rules: "Да, принимаю" }; var options = { method: "post", contentType: "application/json", payload: JSON.stringify(testData), muteHttpExceptions: true }; var response = UrlFetchApp.fetch(BOT_WEBHOOK_URL, options); Logger.log(response.getContentText()); }
