⚔️ Slash-команды Discord
/help
Показать все доступные команды с описанием
/help
/ban <user> [reason]
Забанить пользователя (требуется право BanMembers)
/ban @нарушитель Систематический флуд
/kick <user> [reason]
Кикнуть пользователя с сервера (требуется KickMembers)
/kick @игрок Нарушение правил общения
/mute <user> <time> [reason]
Замьютить пользователя на заданное время (10m, 1h, 1d)
/mute @спам 30m Спам в чате
/create-role <name> [color]
Создать роль с HEX-цветом (требуется ManageRoles)
/create-role VIP #ffd700
/action <user> give_role <role>
Выдать роль пользователю по названию
/action @игрок give_role Участник
/warn <user> <reason>
Выдать предупреждение пользователю с записью в лог
/warn @игрок Оскорбление в чате
/action <user> send_message <text>
Отправить личное сообщение пользователю от имени бота
/action @игрок send_message Добро пожаловать!

ℹ️ О сервере
🎮 GameVerse Server

GameVerse — это живое игровое сообщество, где встречаются люди, объединённые любовью к играм. Мы строим среду без токсичности, с настоящей модерацией и реальными людьми за ней.

🖥️ Хостинг: Railway (сайт + Discord бот) + BotHost (Telegram бот)
🗄️ База данных: PostgreSQL (Railway)
🤖 AI: GPT-4o-mini (OpenAI)
🛡️ Инструкция для модераторов Telegram

Чтобы принимать заявки через Telegram, необходимо пройти авторизацию:

  1. Найди Telegram-бот администрации и напиши /start
  2. Введи команду /auth — бот пришлёт ссылку для входа
  3. Нажми «Войти через Discord» и авторизуйся в браузере
  4. Бот проверит наличие роли Модератор/Admin на Discord-сервере
  5. После подтверждения ты сможешь нажимать ✅/❌ на поступающих заявках

⚠️ Если у тебя нет роли Модератора на Discord — кнопки будут заблокированы.
