🤖 Discord Bot

Модерация, заявки и автоматизация для вашего Discord сервера

🤖 Привет! Я бот-помощник. Ты можешь попробовать мои команды прямо здесь!
/help
Показать список всех команд с подробным описанием
Пример: /help
/ban <user> [reason]
Забанить пользователя на сервере (требует права BanMembers)
Пример: /ban @username Нарушение правил
/kick <user> [reason]
Кикнуть пользователя с сервера (требует права KickMembers)
Пример: /kick @username
/create-role <name> [color]
Создать новую роль с указанным цветом
Пример: /create-role Модератор #ff0000
/action <user> give_role <role>
Выдать роль пользователю
Пример: /action @user give_role Модератор
/action <user> send_message <text>
Отправить личное сообщение пользователю
Пример: /action @user send_message Привет!

🤖 О боте

Этот бот создан для автоматизации процессов на Discord сервере. Он включает в себя систему заявок, модерацию и фильтрацию плохих слов.

📋 Подключение Google Forms

Следуйте инструкции, чтобы связать Google Forms с ботом:

  1. Создайте Google Form с нужными полями
  2. Откройте "Расширения" → "Apps Script"
  3. Скопируйте код ниже и вставьте в редактор
  4. Измените URL на ваш домен
  5. Нажмите "Сохранить" и "Развернуть"
  6. Настройте триггер: "При отправке формы" → "onFormSubmit"
var BOT_WEBHOOK_URL = "https://ВАШ_ДОМЕН/api/applications";

var FIELD_MAP = {
  discord_id:       "Дискорд",
  discord_username: "Ник",
  age:              "Сколько вам лет",
  name:             "Ваше имя (если не хотите писать напишите свой ник)",
  activity:         "Сколько в день вы актив",
  games:            "В какие игры вы играете (не обязал)",
  rules:            "Принимаете ли вы правила сервера?",
};

function onFormSubmit(e) {
  var response = {};
  for (var field in FIELD_MAP) {
    var label = FIELD_MAP[field];
    var value = e.response.getItemResponses().find(
      function(item) { return item.getItem().getTitle() === label; }
    );
    response[field] = value ? value.getResponse() : "";
  }
  
  var options = {
    method: "post",
    contentType: "application/json",
    payload: JSON.stringify(response)
  };
  
  UrlFetchApp.fetch(BOT_WEBHOOK_URL, options);
}
